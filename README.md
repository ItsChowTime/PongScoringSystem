# PongScoringSystem
Python script used to identify and track objects from an external webcam input. To be later connected and drive LED table based on if a specific ping pong ball is sucessfully shot into a cup

ball.py contains script used to detect and track ping poll balls of different colours using an hsv color gradient filter 
cup-detect.py contains script used to detect cups on the table by means of a canny filter and contour mapping
